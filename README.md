Telegram Bot для продажу одягу

Цей проєкт — Telegram-бот, який дозволяє користувачам переглядати каталог одягу та оформлювати замовлення безпосередньо в Telegram.

Особливості:
1. Перегляд товарів за категоріями (штани, кофти, аксесуари тощо)
2. Додавання товарів у кошик
3. Оформлення замовлення
4. Підтримка кнопок меню для зручної навігації

Встановлення та запуск

1) Клонувати репозиторій
git clone https://github.com/ваш_нікнейм/назва_репозиторію.git

3) Створити та активувати віртуальне середовище (рекомендовано)
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

3) Встановити залежності
pip install -r requirements.txt

4) Створити файл .env (для зберігання токена бота)

5) Запустити бота
python bot.py

4) Натискати на товари, щоб переглянути деталі.
